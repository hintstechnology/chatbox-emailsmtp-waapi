# Chatbox Frontend

Frontend aplikasi chatbox untuk customer support.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file (optional):
```
VITE_GRAPHQL_URL=http://localhost:5000/graphql
```

3. Run development server:
```bash
npm run dev
```

Aplikasi akan berjalan di `http://localhost:5001`

## Build

```bash
npm run build
```

Output akan berada di folder `dist/`
